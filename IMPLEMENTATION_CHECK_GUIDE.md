# 実装確認画面ガイド

## 概要

実装確認画面は、CMSシステムの実装状況を確認・テストするための管理画面です。
管理者や編集者向けに、システムの状態を一目で確認できる機能を提供します。

## アクセス方法

1. ダッシュボードにログイン
2. 管理者または編集者権限でアクセス
3. ダッシュボード上部の「📊 実装確認」ボタンをクリック
4. または直接 `/dashboard/implementation-check` にアクセス

## 機能一覧

### 1. 概要タブ

実装状況の全体像を確認できます。

- **大学**: 登録されている大学の数と有効な大学の数
- **手続きフロー**: 総フロー数、入学フロー数、卒業フロー数
- **コンテンツ**: 総コンテンツ数、公開済み数、下書き数
- **手続きステップ**: 全フローに含まれるステップの総数

実装済み機能の一覧も表示され、各機能へのリンクから直接アクセスできます。

### 2. 大学タブ

登録されているすべての大学の一覧を表示します。

各大学カードには以下が表示されます：
- 大学名
- 大学コード
- ドメイン
- 説明
- 有効/無効ステータス

各大学から直接以下にアクセスできます：
- 入学手続きページ
- 手続きフロー一覧ページ

### 3. 手続きフロータブ

すべての手続きフローの一覧を表示します。

各フローカードには以下が表示されます：
- フロー名
- 説明
- フロータイプ（入学、卒業など）
- ステップ数
- 大学ID
- 有効/無効ステータス

各フローから詳細ページに直接アクセスできます。

### 4. コンテンツタブ

すべてのコンテンツの一覧を表示します。

各コンテンツカードには以下が表示されます：
- タイトル
- スラッグ
- 大学ID（関連する場合）
- コンテンツタイプ
- 公開ステータス（公開、下書き、アーカイブ）

各コンテンツから詳細ページに直接アクセスできます。

### 5. 進行状況タブ

実装の進行状況を確認できます。

- **完了した機能**: 実装済みの機能一覧
- **実装済みAPI**: 利用可能なAPIエンドポイント一覧

## 使用方法

### タブの切り替え

画面上部のタブをクリックして、各セクションに切り替えます。

### クイックアクセス

各カードのボタンから、関連するページに直接アクセスできます。

### URLパラメータ

タブを直接指定してアクセスすることもできます：

```
/dashboard/implementation-check?tab=universities
/dashboard/implementation-check?tab=flows
/dashboard/implementation-check?tab=contents
/dashboard/implementation-check?tab=progress
```

## トラブルシューティング

### データが表示されない場合

1. バックエンドサーバーが起動しているか確認
2. データベースに初期データが投入されているか確認
3. ブラウザのコンソールでエラーを確認
4. ネットワークタブでAPIリクエストの状態を確認

### エラーが発生する場合

1. ブラウザのコンソールでエラーメッセージを確認
2. ネットワークタブでAPIレスポンスを確認
3. バックエンドのログを確認
