## Railwayでの設定

### 環境変数の追加

Railwayダッシュボードで以下の環境変数を設定：

1. Railwayダッシュボードにアクセス
2. フロントエンドサービスを選択
3. 「Variables」タブを開く
4. 以下の環境変数を追加：
   ```
   NEXT_PUBLIC_USE_MOCK_DATA=true
   ```

### デプロイ

環境変数を設定後、Railwayが自動的に再デプロイを開始します。
